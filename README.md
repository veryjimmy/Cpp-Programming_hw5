並分為兩個部分：<br/>
1. 判斷輸入的長度是否正確
2. 判斷輸入的身分證是否正確
<br/>
輸入：<br/>
身分證字號<br/>
輸出：<br/>
若正確會顯示 `The ID is right!` 
<br/>
若錯誤會顯示 `Input error!`
<br/>
TestInputFail: 選擇自行輸入身高來測量 BMI ，依序輸入身高 `-20` 公分、體重 `0` 公斤，檢查答案為 `Wrong Input` 。
